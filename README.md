# 정보보안기사 실기 플래시카드

정보보안기사 실기 시험 대비를 위한 웹 기반 플래시카드 학습 도구입니다.

## 주요 기능

- **플래시카드 학습**: 클릭/스페이스바로 카드 뒤집기, 좌우 화살표/스와이프로 이동
- **카테고리 필터**: 위험관리, 시스템 보안, 네트워크 보안, 웹/애플리케이션 보안, 법규 및 관리체계, 침해사고 분석 및 대응
- **유형 필터**: 단답형, 서술형, 실무형, 계산형
- **빈도 필터**: 최빈출, 일반
- **셔플**: Fisher-Yates 알고리즘 기반 랜덤 섞기
- **시험 모드**: 현재 필터 조건에서 랜덤 10문제 추출

## 기술 스택

- **프론트엔드**: 순수 HTML/CSS/JS (프레임워크 없음)
- **마크다운 렌더링**: [marked.js](https://github.com/markedjs/marked) (CDN)
- **정적 서버**: Python3 `http.server`

## 실행 방법

```bash
# 기본 (포트 8080)
./run.sh

# 포트 지정
PORT=3000 ./run.sh
```

## 프로젝트 구조

```
quiz_algisa/
├── index.html          # 메인 HTML (필터 바, 카드 씬, 네비게이션)
├── style.css           # 반응형 CSS (카테고리별 색상, 3D 카드 플립)
├── app.js              # 앱 로직 (필터링, 렌더링, 시험 모드, 이벤트)
├── data.json           # 전체 카드 데이터 (265장)
├── run.sh              # 정적 서버 실행 스크립트
├── quiz_data.md        # 원본 퀴즈 데이터 (마크다운)
├── review_result.json  # QA 리뷰 결과 (삭제/수정 대상 카드 ID)
├── new_data/           # 추가 데이터 원본 및 변환 결과
│   ├── new_data_01.json
│   ├── new_data_02.txt
│   ├── new_data_03.md
│   └── *_converted.json
└── algisa/             # 참고 PDF 자료 (알기사 시리즈)
```

## 조작법

| 동작 | 방법 |

| 다음/이전 카드 | `→`/`←` 키 또는 좌우 스와이프 |

| 시험 모드 | 시험 버튼 (필터 기준 랜덤 10문제) |
